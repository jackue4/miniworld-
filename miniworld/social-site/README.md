# MiniWorld 游戏社交网站原型

这是一个移动端优先的游戏向社交网站首页原型，参考用户提供的图一框架进行重新设计。

## 运行

```bash
npm install
npm run dev
```

打开终端输出的本地地址即可预览。

## 当前包含

- 顶部品牌栏、搜索和通知入口
- 角色形象展示区
- 在线好友横向列表
- Continue Playing 游戏卡片
- Recommended 推荐内容
- 发布/创建浮动按钮
- 底部导航
- 点击卡片、好友、导航、创建按钮会切换状态或打开面板
